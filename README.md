# 🎩 Gent — Monorepo

This repository contains all components of our graduation project in a single **monorepo** setup.  
The system consists of:

- 🚀 **CLI Tool (Node.js)** — A Git-like version control CLI with init, status, add, commit, log, branch, and checkout commands.
- 🐍 **Backend Server (Django)** — REST API backend providing data and authentication.
- 💻 **Web App (React)** — User-facing web interface built with React.
